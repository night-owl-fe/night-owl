## 设计模式的分类
* Creational Design Patterns
    * Constructor(构造器)
    * Factory(工厂)
    * Abstract(抽象工厂)
    * Prototype(原型)
    * Singleton(单例)
    * Builder(建造者)
* Structural Design Patterns
    * Decorator(装饰)
    * Facade(外观)
    * Flyweight(享元)
    * Adapter(适配器)
    * Proxy(代理)
* Behavioral Design Patterns
    * Iterator(迭代)
    * Mediator(中介者)
    * Observer(观察者)
    * Visitor(访问者)
